# Arkanoid Game для Sineys 2024

## Описание проекта

Этот проект представляет собой прототип классической игры Arkanoid, разработанный для хакатона. Игрок управляет платформой, перемещающейся по нижней части экрана, отталкивая шарик для разрушения блоков. Цель — уничтожить все блоки на уровне и не дать шарику упасть за пределы игрового поля.

## Основные особенности

- Управление платформой с помощью клавиатуры или мыши.
- Шарик отскакивает от платформы, стен и блоков.
- Разрушаемые блоки.
- Поражение при падении шарика за пределы игрового поля.
- Дополнительная игровая механика для разнообразия (бустер на удваевание шариков).

## Запуск игры

1. Скачайте или клонируйте репозиторий:
   ```bash
   git clone https://github.com/aleverdes/sineys-arkanoid/
2. Запустите проект на платформе Windows в полноэкранном режиме.

## Управление
- Клавиатура: Используйте стрелки для перемещения платформы.
- Мышь: Двигайте мышь для управления платформой. 
- ЛКМ: для запуска шарика.
- Пробел: Запустить шарик.
- Esc: Пауза/выход в меню.

## Дополнительная информация
- В игре реализован базовый графический интерфейс (UI) и меню.
- Игра адаптирована для разных разрешений экранов.

# Arkanoid Game for Sineys 2024

## Project Description
This project is a prototype of the classic Arkanoid game, developed for a hackathon. The player controls a paddle that moves horizontally at the bottom of the screen, bouncing a ball to destroy blocks. The goal is to clear all blocks on the level and prevent the ball from falling out of bounds.

## Key Features
- Player-controlled paddle with keyboard or mouse input.
- Ball bounces off the paddle, walls, and blocks.
- Destructible blocks.
- Game over occurs when the ball falls out of bounds.
- Additional gameplay mechanic for variety (booster for double count of the balls).

## How to Run
1. Download or clone the repository:

```bash
git clone https://github.com/aleverdes/sineys-arkanoid/
```
2. Run the game on Windows in fullscreen mode.

## Controls
- Keyboard: Use arrow keys to move the paddle.
- Mouse: Move the mouse to control the paddle.
- Left Mouse Button: Launch the ball.
- Space: Launch the ball.
- Esc: Pause/Exit to the menu.

## Additional Information
- Basic graphical user interface (UI) and menu are implemented.
- The game is adapted for different screen resolutions.
